angular的使用

什么是angular

  • AngularJS 是一个 JavaScript 框架。它可通过 \ 标签添加到 HTML 页面。
  • 学习曲线 - 入门容易 - 深入较难
  • 框架 - 框架是一种特殊的、已经实现了的WEB应用，你只需要对它填充具体的业务逻辑。这里框架是起主导作用的，由它来根据具体的应用逻辑来调用你的代码。框架有：knockout、sproutcore等。 我们按照人家的规则写好了代码，被动性质的，好比对方有房子，而我们只需要放入家具即可；
  • 类库 - 类库是一些函数的集合，它能帮助你写WEB应用。起主导作用的是你的代码，由你来决定何时使用类库。类库有：jQuery等
  • MVVM/MVC
    • MVC：
      • model(模型) view(视图) controller(控制器)
      • 流程：view(人为操作) –》 controller –》 model –》 view
    • MVVM：
      • model(数据模型) view(视图) viewModel

        强约束

使用angular

安装angular

  • 先初始化 package.json文件

  • 安装angularJs

在当前的文件夹下生成node_modules文件将安好的内容放入（名字(package.json)不能叫angular1）

运用angular

  • AngularJS 通过被称为指令的新属性来扩展 HTML，带有前缀 ng-，通过内置的指令来为应用添加功能，允许自定义指令。

  • ng-app

    • 初始化一个angular应用程序,让angular自行启动，并且创建一个$rootScope跟作用域(window)

  • ng-model(表单input元素上)
    实现双向数据绑定，如果有数据会将数据绑定到输入框中，没数据会在当前作用域下声明，如果改变了输入框的内容，会影响数据的变化，如果数据变化了，会将新的内容绑定到输入框中

  • ng-init
    初始化数据

  • 取值表达式
    { { } }，取当前作用域上得值，支持运算、赋值、三元表达式。

    用点的形式取的就是.前边对象的属性，如果这个对象不存在，则声明

防止闪烁

  • ng-bind 和 ng-clock
    • ng-bind用法同{ { } } 防止单个元素闪烁
    • ng-clock 需要自定义样式 将带有ng-clock属性的元素隐藏掉

控制器和模块

  • 生成模块

  • 创建控制器

  • 应用控制器

控制器的特点

  • 1.主要控制数据
  • 2.不要在控制器中控制DOM
  • 3.控制器的作用范围(与DOM标签结构平齐)
  • 4.控制器可以嵌套(嵌套的是带有控制器的DOM节点，不是控制器本身)
  • 5.有继承关系，父作用域不能继承子作用域，子作用域可以继承父作用域

配置全局属性

过滤器

过滤器（filter）正如其名，作用就是接收一个输入，通过某个规则进行处理，然后返回处理后的结果。
分别是：currency(货币)、date(日期)、filter(子串匹配)、json(格式化json对象)、limitTo(限制个数)、lowercase(小写)、uppercase(大写)、number(数字)、orderBy(排序)，总共九种。
除此之外还可以自定义过滤器，可以满足任何要求的数据处理。
